eraメガテン__キャラ引継ぎ一括選択パッチ
*******************************************************************************

■ 始めに
当パッチは周回ニューゲームにおけるキャラ引継ぎ時に引継ぎ可能なすべてのキャラを
一括で引継げるようにするパッチです。

■ 内容物
・readme           (当ファイル)
・SHOP_NEWGAME.ERB (一部修正)

■ 謝辞
eramaker、Emuera、及びeraMegatenに関わったすべての方々への多大なる謝辞をこの場で申し上げさせて頂きます。
また該当モジュールの制作者様にも重ねて謝辞を申し上げさせて頂きます。
旅人様のreadmeの書き方がとてもすばらしかったためパk..参考にさせていただきました。
制作者: 黒天

-------------------------------------------------------------------------------
１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.283

■ 導入
下記の通り、ERBフォルダ配下の対象フォルダ内に、
同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

[フォルダ構成図]
+[ERB]
|
+--+[ＳＨＯＰ関連]
   |
   +--+[SHOP_NEWGAME.ERB]

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■ 引継ぎキャラ選択時に
「 [0] 引継ぎ可能なすべてのキャラの引継ぎを変更する。」
を選択することで引継ぎ可能なキャラすべての引継ぎ方法を選択した引継ぎ方法に変更します。

(作者コメント)
0.283の取り込みパッチ一覧にあったけどパッチ1277(イベント出現告知パッチ)の修正で
上書きされて消えてるっぽいので0.283版をリリース
修正内容自体は0.282+修正ファイル版と同じです。

-------------------------------------------------------------------------------
３．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法
当パッチではフラグは使用していません。
また修正箇所が分かる様、修正ファイルには下記の様に修正コメントが存在します。

;---- EDIT 002 ADD START --------
	CALL TEST_ABC
;---- EDIT 002 ADD END ----------

「EDIT 002」等で検索すれば、手を加えた全ての修正箇所が分かります。
前の処理に戻したい場合は上記コメントで囲まれた箇所の削除と
ADDがMODになっているコメントで囲まれている行を以下の通り変更してください。
<647行目>
変更前）ELSEIF (RESULT < 0 || RESULT >= CHARANUM || FINDQ(RESULT) == -1) && RESULT != 0
                           ↓
変更後）ELSEIF RESULT < 0 || RESULT >= CHARANUM || FINDQ(RESULT) == -1

<669〜679行目>
677行目を除き削除してください。
