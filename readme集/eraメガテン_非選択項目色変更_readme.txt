*******************************************************************************
eraメガテン_非選択色変更パッチ
*******************************************************************************

■ 始めに
当パッチは調教のフィルタリング項目の非選択状態(フィルタ有効)時の色と
調教メニュー登録画面での非表示状態時の色を変更するためのパッチです。


■ 内容物
・readme              (当ファイル)
・SYSTEM_FUNCTION.ERB (一部修正)
・COMREGISTER.ERB     (一部修正)
・USERCOM.ERB         (一部修正)

■ 謝辞
eramaker、Emuera、及びeraMegatenに関わったすべての方々への多大なる謝辞をこの場で申し上げさせて頂きます。
また該当モジュールの制作者様にも重ねて謝辞を申し上げさせて頂きます。
旅人様のreadmeの書き方がとてもすばらしかったためパk..参考にさせていただきました。
制作者: 黒天

-------------------------------------------------------------------------------
１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.288+修正ファイル3

■ 導入
下記の通り、ERBフォルダの配下にある
対象フォルダ内に、同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

[フォルダ構成図]
+[ERB]
|
+--+[SYSTEM_FUNCTION.ERB]
|
+--+[調教関連]
   |
   +--+[COMREGISTER.ERB]
   |
   +--+[USERCOM.ERB]

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■ 式中関数「COLOR()」に"light-gray"(明灰色)を追加
　 調教時のフィルタリング項目の非選択状態時および
　 調教メニュー登録画面の表示/非表示の非表示に上記追加色を使用するように変更

-------------------------------------------------------------------------------
３．作者の戯言
-------------------------------------------------------------------------------
ちょっと非選択・非表示時の文字が見え辛すぎたので少し明るめにしました。
一応デフォルト文字色よりは暗いので見間違えることはないと思う。。。

本当は現状の"gray"(灰色)を"dark"(闇色)とかに変更して
今回追加した色を"gray"に当てはめたいなと思っていたのですが。。。
それをやると変更箇所がとんでもないことになりそうだったので断念
明灰色とかなんかしっくりこないけど、他にいい名前も思いつかずごらんの有様に

-------------------------------------------------------------------------------
４．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法
本パッチではフラグの追加などは行っておりません。

修正箇所については0.288+修正ファイル3環境を別の場所に構築し、本パッチに含まれている
ファイルと同名のファイルと内容を比較して確認してください。
前の処理に戻したい場合もお手数ですが
0.288+修正ファイル3の環境を別の場所で展開し、該当箇所の差分を比較し修正してください。
