*******************************************************************************
eraメガテン_能力上昇キャラ切替
*******************************************************************************

■ 始めに
当パッチは能力の上昇において上昇能力選択画面から対象キャラを前のキャラか次のキャラに
切替可能にするためのパッチです。


■ 内容物
・readme  (当ファイル)
・ABL.ERB (一部修正)

■ 謝辞
eramaker、Emuera、及びeraMegatenに関わったすべての方々への多大なる謝辞をこの場で申し上げさせて頂きます。
また該当モジュールの制作者様にも重ねて謝辞を申し上げさせて頂きます。
旅人様のreadmeの書き方がとてもすばらしかったためパk..参考にさせていただきました。
制作者: 黒天

-------------------------------------------------------------------------------
１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.285

■ 導入
下記の通り、ERBフォルダの配下にある「調教関連」配下の
対象フォルダ内に、同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

[フォルダ構成図]
+[ERB]
|
+--+[調教関連]
   |
   +--+[ABL]
      |
      +--+[ABL.ERB]

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■ 
能力上昇画面で選択肢に[前のキャラ]、[次のキャラ]を追加
選択することで一覧画面に戻らず前のキャラ、次のキャラの能力上昇画面に移動する。

(作者コメント)
調教終了時の能力上昇画面でも[前のキャラ]、[次のキャラ]が表示されてしまいます。
消したかったけど、よく分からないし押してもキャラが切り替わらないみたないなので
放置することにしました。

-------------------------------------------------------------------------------
３．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法
本パッチではフラグや関数の追加などは行っておりません。

前の処理に戻したい場合はお手数ですが
0.285を別の場所で展開し、該当箇所の差分を比較し修正してください。
