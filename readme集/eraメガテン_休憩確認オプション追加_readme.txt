*******************************************************************************
eraメガテン_休憩確認オプション追加パッチ
*******************************************************************************

■ 始めに
当パッチはSHP画面で[休憩]を選択した時の確認有無をオプションで設定するためのパッチです。
また、確認有りの場合に「朝まで休憩する」選択肢を追加します。


■ 内容物
・readme           (当ファイル)
・FLAG.csv         (一部追加)
・SYSTEM.ERB       (一部修正)
・SHOP_REST.ERB    (一部修正)
・OPTION.ERB       (一部修正)

■ 謝辞
eramaker、Emuera、及びeraMegatenに関わったすべての方々への多大なる謝辞をこの場で申し上げさせて頂きます。

-------------------------------------------------------------------------------
１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.288+修正ファイル3

■ 導入
下記の通り、ERB,CSVフォルダの配下にある
対象フォルダ内に、同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

[フォルダ構成図]
+[CSV]
|
+--+[FLAG.csv]

+[ERB]
|
+--+[SYSTEM.ERB]
|
+--+[ＳＨＯＰ関連]
|  |
|  +--+[SHOP_REST.ERB]
|
+--+[調教関連]
   |
   +--+[OPTION.ERB]

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■ 休憩選択時に休憩するかどうかの確認をするかしないか
　 設定するためのオプションが追加されます。
　 また、オプションが確認するように設定されている場合、
　 休憩選択肢に休憩する・しないの他に「朝まで休憩する」が追加されます。
　 「朝まで休憩する」を選択した場合、次の日の朝まで自動的に休憩するが選択された
　 場合の処理が行われます。

-------------------------------------------------------------------------------
３．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法
本パッチではFLAG.csvの630番に休憩確認フラグを追加しています。
本パッチを導入した環境でセーブをするとセーブデータに休憩フラグの値が保存されるように
なるためご注意ください。

修正箇所を確認したい場合は、お手数ですが0.288用修正ファイル3を解凍し
同名ファイルと内容を比較してください。
前の処理に戻したい場合もお手数ですが
0.288用修正ファイル3を別の場所で展開し、該当箇所の差分を比較し修正してください。
