･6/23
Zoma強化の方は今回ので元に戻しましたが、原作仕様のままだと調教関係やMAG強化で天井知らずに強くなり過ぎるので
今後は弱体化させたほうがいいかもしれません(強化一回ごとに強化を2/3にするとか･･･)

もしかしたら、改造はしないかもしれません

姿がすでに変わってるのに、耐性が変化しないのは変だと思うので合体即変化にしてます。
