戦闘口上パッチりーどみー

□　定型文
　パッチの適用は自己責任でお願いします。
　最悪セーブデータの破損等が起こるおそれもありますので、
　適用前に適宜バックアップをお忘れなく。


□　概要
　口上セレクト機能を利用した専用口上を、戦闘時の口上にも反映させるためのパッチです。
　タマモ及びハム子の専用口上のみで動作を確認しています。

□　改変した箇所
　・ERB\RPG\戦闘\SYSTEM_BATTLE.ERB
　　1484行目以降の、ＣＯＯＰ発生者、ＣＯＯＰ参加者の口上を呼び出す部分について
　　該当者のCFLAG:口上セレクトが０以外の場合は専用口上を読みに行くよう修正

　・ERB\口上\KOJO_RPG.ERB
　　255行目以降の攻撃者の戦闘口上呼び出しについて、
　　攻撃者のCFLAG:口上セレクトが０以外である場合は専用口上を読みに行くよう修正
