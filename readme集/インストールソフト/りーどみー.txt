パッチの利用法
簡単なのはインストールソフト追加パッチ全部盛りをコピペです。
このパッチのソフトが全部追加されます。

もしくはソフトの追加パッチとそれぞれの効果発揮パッチを個別にコピペしてください。
物はパッチの中身フォルダにあります。
コピペしたソフトだけ動きます。
ただしソフトの追加パッチを入れないで個々のソフトパッチを入れるとエラー落ちします。
ご注意ください。

ダブルバリューの効果もうちょっと増やしてみてもいいかなあ。
