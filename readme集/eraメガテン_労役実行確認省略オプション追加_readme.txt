*******************************************************************************
eraメガテン_労役実行確認省略オプション追加パッチ
*******************************************************************************

■ 始めに
当パッチは食堂・売春において実行の確認を省略することのできるオプションを追加し
労役設定時に毎回実行有無を選択する必要がないようにするためのパッチです。


■ 内容物
・readme           (当ファイル)
・Flag.csv         (一部追加)
・OPTION.ERB       (一部修正)
・PROSTITUTION.ERB (一部修正)
・RESTRANT.ERB     (一部修正)

■ 謝辞
eramaker、Emuera、及びeraMegatenに関わったすべての方々への多大なる謝辞をこの場で申し上げさせて頂きます。

-------------------------------------------------------------------------------
１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.281R+修正ファイル3

■ 導入
下記の通り、CSVフォルダの配下にある「Flag.csv」ファイル
およびERBフォルダの配下にある「調教関連」配下の
対象フォルダ内に、同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

[フォルダ構成図]
+[CSV]
|
+--+[Flag.csv]

+[ERB]
|
+--+[調教関連]
   |
   +--+[OPTION.ERB]
   |
   +--+[労役]
      |
      +--+[PROSTITUTION.ERB]
      |
      +--+[RESTRANT.ERB]

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■ 労役実行確認省略設定のオプションを追加しています。
上記オプションにいおいて「省略する」を設定した場合、
食堂／売春において労役者を選択した場合に発生する労役実行確認を省略し
無条件で労役するようにします。
これにより多数の奴隷を労役に従事させる場合の労力が減ると思われます。

-------------------------------------------------------------------------------
３．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法
当パッチでは労役実行確認を省略するかどうかのフラグを追加しています。
FLAGの614番を使用しています。
また修正箇所が分かる様、修正ファイルには下記の様に修正コメントが存在します。

;---- EDIT 労役実行確認省略オプション追加(2011/02/04) ADD START ---------
	CALL TEST_ABC
;---- EDIT 労役実行確認省略オプション追加(2011/02/04) ADD END -----------

「EDIT 労役実行確認」等で検索すれば、手を加えた全ての修正箇所が分かります。
前の処理に戻したい場合はお手数ですが
0.281Rを別の場所で展開し、該当箇所の差分を比較し修正してください。
