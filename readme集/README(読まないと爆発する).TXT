・追加されるスキルは以下の4種です
マイト		　　　MAGIC　敵1体を爆弾に変える。スキル系統は。
マイトマ	　　　MAGIC　敵1列を爆弾に変える。スキル系統は。

・効果が変わるスキルは以下の2種です
パトラ		BOMBを治療可能に
ペンパトラ	BOMBを治療可能に
効果は変更していませんが、リカームドラ・アムリタ等でも治療が可能です

・追加されるステートは以下の1種です
BOMB		：火炎・核熱相性の攻撃で引火爆発する爆弾になる
			　火炎・核熱相性の攻撃あるいはBOMBの爆発でダメージを受ける事が発動のトリガー。
			　発動後自身と隣接1マスのキャラクターに発動者の最大HP比例のダメージを付与する
			　このダメージによるCOOP、1moreは発生しない。

　＞状態BOMBがソウルハッカーズ発祥だと知り猛省しているようだ…


BOMBはデビルチルドレンがオリジナルだと思ってました、申し訳ない。

簡単に説明するなら、自分ではあまり使わないけど敵が使ってくると辛いタイプの異常です。
ハマ・ムド系に近いですが本人以外も巻き込む事がポイント。
対策は神経・火炎・核熱耐性を高める事、無効化や反射特技を使う事
パトラ・ペンパトラでさっさと回復等等。
何気にパトラ系の地位向上に役立つんじゃないでしょうか。必要かはさておき。
本来はボムディという専用回復呪文があるんですが今回は省きました。

強い気もするけど
＞ザコ対策ならハンマ・ムド系の方が有効性が高い
＞COOP狙った方がダメージ効率が良い
＞強い奴にはなかなか当たらない
＞BOMB化確認してから火炎攻撃、と二度手間
＞かといって1ターンで発動させようとすると運が絡む
＞本体ダメージはさておき、拡散ダメージは高いとは言い難い
とやっぱり使いにくい。

それはそれとして、BOMB状態ってこのままでいいのか悩んで居たりします。
原作での状態名は思い出せないし、マイトはそのまま英語に戻すとおかしな事になるし。
CAUTIONやDANGERだと引火との関連性が薄くなってしまうような気もするジレンマ。
EXPLOSIVEは長すぎる。やはりBOMBか、ボムなのか！
→BOMB出よかったようです(ソウルハッカーズ的に考えて)
