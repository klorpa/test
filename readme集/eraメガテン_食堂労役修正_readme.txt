■ 始めに
当パッチは食堂において接客人数制限処理の発生条件を仕様通りに修正および
接客人数制限時の体力消費量を修正しています。


■ 内容物
・readme       (当ファイル)
・RESTRANT.ERB (一部修正)

２．機能
-------------------------------------------------------------------------------
■ 
・接客人数の制限処理発生条件を仕様どおり 現在体力が最大体力の半分以上である場合に修正
・現在体力と消費体力が同じだった場合に体力が尽きるのに接客人数の制限が行われない
　可能性があったのを修正
・接客人数が制限された場合に消費体力を制限された接客人数に相応の量になるように修正
　(それでも体力が尽きる場合は今までどおり体力が1残るように消費体力を設定)
・体力尽きてダウンしても働かせ続けるのは心が痛むので労役を解除するよう修正
・サラリー計算の最低保証労役単価の処理が修正されていなかったので修正

(作者コメント)
客制限が仕様どおりになると基本的な1日あたりの体力回復量が約45％なので
体力尽きないようにするためには一日置きの週3で働かせるほうがいいかもしれない。
