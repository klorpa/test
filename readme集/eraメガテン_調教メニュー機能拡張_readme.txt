*******************************************************************************
eraメガテン_調教メニュー機能拡張パッチ
*******************************************************************************

■ 始めに
当パッチはキャラ毎に保存される調教メニューを追加および
一部調教メニュー関連の機能を変更するためのパッチです。

■ 内容物
・readme           (当ファイル)
・CSTR.csv         (一部修正)
・COMREGISTER.ERB  (一部修正)
・USERCOM.ERB      (一部修正)

■ 謝辞
eramaker、Emuera、及びeraMegatenに関わったすべての方々への多大なる謝辞をこの場で申し上げさせて頂きます。
また該当モジュールの制作者様にも重ねて謝辞を申し上げさせて頂きます。
旅人様のreadmeの書き方がとてもすばらしかったためパk..参考にさせていただきました。

-------------------------------------------------------------------------------
１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.290
ver0.290+修正ファイル1
ver0.290+修正ファイル2
ver0.290+修正ファイル3

■ 導入
下記の通り、CSVおよびERBフォルダの配下にある
対象フォルダ内に、同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

[フォルダ構成図]
+[CSV]
|
+--+[CSTR.csv]

+[ERB]
|
+--+[調教関連]
   |
   +--+[COMREGISTER.ERB]
   |
   +--+[USERCOM.ERB]

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■ 調教メニュー登録に以下の機能を追加
・キャラ毎に保存される調教メニューを3件追加
　登録された調教メニューの内容および名称はCSTR:71〜CSTR:73に保存されます。
・調教メニューの[表示/非表示]設定で1行に表示するメニュー数を3件から5件に変更
・連続登録に「繰り返し」を追加
　繰り返しの選択した箇所の前までのコマンドを繰り返し実行するように登録します。
・調教メニュー名称を空で入力した場合、名称変更がキャンセルされるように変更

-------------------------------------------------------------------------------
３．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法
本パッチではキャラ毎の調教メニュー保存先としてCSTR:71〜CSTR:73を追加しています。
セーブデータに保存されるため、導入には注意をお願いします。

修正箇所を確認したい場合は、お手数ですが0.290を別の場所へ展開し、
同名ファイルと内容を比較してください。
前の処理に戻したい場合もお手数ですが0.290を別の場所で展開し、
該当箇所の差分を比較して修正してください。
