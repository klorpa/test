□内容

□動作上の変化
1.アナル奉仕強制、サド・反抗的・プライド高低・抵抗が共通部分とあわせて二重取りになっていたので封印
2.コンドーム精飲、ふたなりかどうかを判定・ペニスの汚れを判定はまぁカット
3.ダブルパイズリフェラなどの助手と奴隷両方が絡むコマンド、相性の補正を奴隷-主人*奴隷-助手にしてみる
4.シックスナインに精液中毒3を追加してみる
5.キャラ間の恋慕を取る関数を作成したのに合わせて、調教者と奴隷の間もそれで判定
　要するに、今までは恋慕は主人調教のときだけ判定にでてたけど
　愛NTR状態なら、NTRってるキャラででて、主人ででなくなる
　ただ、NTR状態は親愛と恋慕の区別をつけないので、元の陥落度合いでわけることに
　好感度の絶対量とかでもいいきがするので要調整項目
6.男嫌いの箇所を女嫌いも同時に判定するように
7.娼婦・淫乱で潰れない快感の否定などを撤廃。
