売却、助手の条件と陥落素質の進捗状況をリアルタイムに確認できる機能の追加パッチです
条件が何だったかすぐ忘れてしまう人用パッチです
能力の上昇画面で、珠一覧の下にでます（条件を全部満たしている場合は何も出ません）

ただ表示が煩雑なのと、若干ネタバレっぽい感じもするので、オプション化した方がいいかもしれません
グローバル変数は使用していないので、使用の有無はいつでも切り替える事ができます

条件は実際に付加処理をしているERBを元に作りました
条件のうち、依存度に関しては主人に覚素質が無いと、達成の有無はわからないようにしてあります

恋慕系のAbnormalEXPと服従系の感覚合計は、一度不達成になると取り返しがつかないので、黄色と赤で表示しています
ついうっかりおっぱいを鍛えると赤表示で教えてくれるので安心です

陥落素質を持っていると上位素質への条件が表示されるようになります

ABL.ERBの@SHOW_ABLUP_SELECTから呼び出して実体はSHOW_STATUS_SPOILER.ERBにあります
追加部分の他バリアントへの移植や改変等は自由ですが、条件はeraMegaten用なので適当に書き換えて使ってください
