デビルサマナーの大学後のダンジョン追加してみた

依頼形式にしてもよかったけど地の文で挫折、とりあえず攻略はできるよ状態
基本、異界化してるのでクリアしたら入れません


氷川神社はダンジョン的にちょっとショボイ気がする
原作じゃボス無しだし、笠置山と自宅周辺をまとめてみてもいいかも
とりあえずストリゲスとストリゴイイ配置
ミラーがないのでほぼパチモン状態

博物館等は飛ばしました、予定無し

中ボス倒すとボス弱体化、こっちが標準、ちょっと弱めに設定

オルゴンゴースト（強）はちょっと強めに設定
倒したらボーナスあってもいいかも
