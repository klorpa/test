*******************************************************************************
eraメガテン_キャラソート挿入機能追加パッチ
*******************************************************************************

■ 始めに
当パッチは以下の0.299+修正ファイル2環境においてキャラソート機能に
挿入によるキャラの位置変更機能を提供するためのパッチです。

問題1. 

■ 内容物
・readme         (当ファイル)
・CHARA_SORT.ERB (一部修正)

■ 謝辞
eramaker、Emuera、及びeraMegatenに関わったすべての方々への多大なる謝辞をこの場で申し上げさせて頂きます。
また該当モジュールの制作者様にも重ねて謝辞を申し上げさせて頂きます。
旅人様のreadmeの書き方がとてもすばらしかったためパk..参考にさせていただきました。

-------------------------------------------------------------------------------
１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.299+修正ファイル2

■ 導入
下記の通り、ERBフォルダの配下にある
対象フォルダ内に、同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

[フォルダ構成図]
+【ERB】
|
+--+【ＳＨＯＰ関連】
   |
   +--+[CHARA_SORT.ERB]

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■ 追加機能
SHOPコマンド[666]キャラソートにキャラ挿入による位置変更機能を追加
挿入によるキャラ入れ替えを行った場合一人目に選択したキャラが
二人目に選択したキャラの前に挿入されます。

■ 機能確認状況
・0.299+修正ファイル2環境で当パッチを当てた状態でのeraMegatenの起動確認済
・既存のキャラ入れ替えによるソートが正常に動作していることを確認
・挿入ソート時に一人目に選択したキャラが二人目に選択したキャラの
　前の位置へ移動していることを確認

-------------------------------------------------------------------------------
３．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法
本パッチではフラグの追加などは行っておりません。

修正箇所についてはお手数ですが0.299+修正ファイル2環境を別の場所に展開し、
本パッチに含まれているファイルと同名のファイルと内容を比較して確認してください。
前の処理に戻したい場合もお手数ですが0.299+修正ファイル2環境を別の場所に展開し、
同名ファイルの差分を比較し修正してください。
