	ver0.266
	・COOPの威力を使用したスキルの威力に依存するように
	　物理スキルは威力の1/10(最大10)+100を超えている分の1/2、魔法スキルは威力分を命中する度にCOOPフラグに加算し、
	　合計を基本威力として、×（人数・属性一致による倍率補正）×（参加者平均の攻撃力・魔法威力）/（対象の防御力・魔法効果）
	　倍率補正は、100％～600％。ただし、元の攻撃が弱点であることを加味すると、元のダメージに対する比率は200弱点で半分～3倍、150弱点で66％～4倍くらい。
	　クリティカルによる強引なＣＯＯＰは威力半減、ただし弱点＋クリティカルなら威力1.5倍になるように

	　通常攻撃はともかく、中盤以降の銃によるＣＯＯＰ狙いの価値が激減した＞威力に銃攻撃力の1/5を加算するようにしてみる。
	　というか力で威力ガンガン伸びるＳＷＯRDとは違って威力伸び辛いから、マシンガン/アサルトライフル系の攻撃回数を原作並みにエキセントリックにしても良い気がしてきた。
	　敵が単体だと強いけど一杯いると攻撃範囲広くて威力・命中高めのショットガンや狙撃しやすくて特殊弾多いハンドガンに比べて使いにくい感じで。

	・ＣＯＯＰのダメージ計算時、発動者の攻撃力と魔法威力が加算されていなかったのを修正
	・ディア・メディアの威力を1.5倍にしてみる
