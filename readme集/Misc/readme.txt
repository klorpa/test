
【概要】
このゲームは女神転生シリーズの二次創作作品であり、
佐藤敏氏の開発した「erakanon」の改変である「eratohoA」を改変したものとなっています。
尚、ゲームの内容については未完成のものとなっています。

※Emuera専用となっていますので、eramakerでは起動できません。

※修正ファイルによるパッチではなく本体そのものがバージョンアップした場合、絶対に以前のバージョンの本体に上書きはしないで下さい。
　ファイル構成の変更により正常に動作しなくなることがあります



バグ報告・意見要望などありましたら是非本スレなどにご連絡ください。
また、スキル・アイテム・装備・悪魔等のデータを作成した場合、うｐろだにパッチとして上げて下さると幸いです。

本バリアントの改変・改造や、パッチの作成、および機能の移植などについては特に制限はありません。
ご自由にご利用下さい。
