EVENT_MESSAGEに以下のものを追加
;　　70:陰唇・胸同時露出(調教対象)
;　　71:陰唇・胸同時露出(主人)
;　　72:陰唇・胸同時露出(助手)
;　　73:陰唇出(調教対象)
;　　74:陰唇露出(主人)
;　　75:陰唇露出(助手)
;　　76:胸露出(調教対象)
;　　77:胸露出(主人)
;　　78:胸露出(助手)

※陰唇はペニスも含む
　胸の露出は、オープンオッパイ機能で呼び出すものとは別。こちらは44番で呼ばれる
　すべて調教対象の口上なので、74なら、主人がチンコをボロンしてきたときの奴隷のリアクションとなる

@CLOTHES_MESSAGE_KX, ARG, ARG:1, ARG:2を追加
ARGは着脱があった部位
ARG:1は1なら着せた。0なら脱がせた
ARG:2は着脱したキャラ(MASTER/TARGET/ASSIのいずれかが入る)
上記のイベントメッセージで不足な場合、所持してるフラグを駆使して、構築してください

なお、CALL CLOTHES_MESSAGE,ARG,ARG:1,ARG:2をやらないと、着せた・脱がせたのメッセージが入らないので注意してください


動作テスト用のあかりちゃんも同梱しておきます
