戦闘画面に相性を表示するパッチ
for eraMegaten ver0.254[era4660] + 修正ファイル2[era4668]

１．なにこれ
戦闘画面の敵ステータスの下に弱点とか耐性とか表示します
攻撃手段選んで対象を選択する時にその攻撃手段に対する相性を表示します
（ＣＯＭＰ持ち健在・アナライズ可能な敵のみ）


２．なにがしたいの
いちいちANALYZEで相性チェックするのめんどくせえ
どうせノーコストで見れるんだからデビルサバイバーやストレンジジャーニーみたいに
戦闘画面に弱点とか表示させちまえよ

ということで作った。

３．ＦＡＱ

Ｑ．GiboEyesとどう違うの
Ａ．GiboEyesはアナライズ不可（解析度低い）の敵にも効果があります
　　このパッチはANALYZEコマンドで見れる情報を表示してるだけなので、ＣＯＭＰ持ち全滅の時や解析度不足の敵には効果を発揮しません。

Ｑ．見た目が気に入らない
Ａ．君の出番だ。(ｷﾘｯ


４．今後の課題
・画面のデザイン・色のチョイス
　もうちょっとカッコよくできないものか
・コンフィグによる表示方法の切り替えなど
　弱点のみ表示とか無効化以上のみ表示とかしたら
　表示が少なくなってスッキリするんじゃないかなとか思ったり
・色分け以外の表示方法
　[弱]とか[反]とかだとスペースを倍使うので色分けで表示したが
　色のチョイスとか色盲の人どうするよ？とかそんな感じ
・味方の入れ替えの時も表示したらよくね？
　時間無いので後回し
・やる気
　上に挙げた問題点誰か何とかしてくれないかなーﾁﾗｯﾁﾗｯ

５．何とかしてくれる人へ
FLAG:600がONOFFフラグ（仮）になってます。
戦闘画面にONOFF切り替えの項目（デバッグ用）があります。
（仮）なので本体へ取り込みたい場合は適当な所へ移動してください。
