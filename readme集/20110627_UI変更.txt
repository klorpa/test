□内容
1.セットアップ画面で、キャラをクリックした場合(21~26)、そのキャラのステータス画面に飛ぶように
2.フォーメーションの変更を個別入れ替えを基本にし、総入れ替えをする場合も一つ戻ることが出来るように
3.ステータス画面から装備を変更できるように
4.探索中のステータス表示のキャラをクリック(01~06)でそのキャラのフィールド魔法にショートカットされるように
5.単体魔法を使ったあと、キャラをクリック(21~26)で連続してその魔法を使えるように
6.フィールド魔法から悪魔変身を実行出来るように変更
7.戦闘中を示すTFLAG:戦闘中を作成
8.BATTLEで、RESULTがスキップされてる場合、一部処理が通らなかったのを修正し
　勝利処理をBATTLE_ENDとして纏める
