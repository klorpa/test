*******************************************************************************
eraメガテン_EVENT_M修正パッチ(衣装関係)
*******************************************************************************

■ 始めに
当パッチはeraメガテンのイベント管理ファイルの一つ、EVENT_Mを修正するパッチです。
下記の点を修正します。

・衣装名が正しく表示されない不具合

■ 内容物
・readme             (当ファイル)
・EVENT_M.ERB        (一部修正)

■ 謝辞
eramaker、Emuera、及びeraMegatenに関わったすべての方々への多大なる謝辞をこの場で申し上げさせて頂きます。
また該当モジュールの制作者様にも重ねて謝辞を申し上げさせて頂きます。
制作者：旅人

-------------------------------------------------------------------------------
１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.281R 以上

■ 導入
下記の通り、ERBフォルダの配下にある「調教関連」配下の
対象フォルダ内に、同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

[フォルダ構成図]

+[ERB]
|
+--+[調教関連]
   |
   +--+[EVENT_M.ERB]


-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■ 衣装名の表示を修正
調教開始時などに表示される、衣装名を含んだ情景メッセージが正しく表示される様に修正。


■ 実際にやった事
下記の様なPRINTFORM命令を、衣装名が表示される様に修正。
(@CLOTHES_NAME関数は、内部でPRINTを行っている為、呼び出すだけで表示される)

旧）
	PRINTFORM CALL NAME:CLOTHES_NAME,5,TARGET

新）
	CALL CLOTHES_NAME,5,TARGET


--------------------------------------------------
(作者コメント)
『PRINTFORM CALL NAME:CLOTHES_NAME〜』という記述を修正。

-------------------------------------------------------------------------------
３．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法
当パッチはフラグ類は使用していません。後を汚さず着脱可能となっています。
また修正履歴が分かる様、修正ファイルには下記の様に修正コメントが存在します。

;---- EDIT 002 ADD START -------------------------
	CALL TEST_ABC
;---- EDIT 002 ADD END ---------------------------

「EDIT 002」等で検索すれば、手を加えた全ての修正箇所が分かります。
（「ADD」内のソースを全て消し、「DEL」内のコメントアウトを元に戻せば、修正前に戻ります。）
