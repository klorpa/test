*******************************************************************************
eraメガテン_INPUT_YNスペース対応パッチ
*******************************************************************************

■ 始めに
当パッチはINPUT_YN関数で処理される二択選択肢で
スペースキーでも選択できるようにするためのパッチです。


■ 内容物
・readme              (当ファイル)
・SYSTEM_FUNCTION.ERB (一部修正)

■ 謝辞
eramaker、Emuera、及びeraMegatenに関わったすべての方々への多大なる謝辞をこの場で申し上げさせて頂きます。
また該当モジュールの制作者様にも重ねて謝辞を申し上げさせて頂きます。
旅人様のreadmeの書き方がとてもすばらしかったためパk..参考にさせていただきました。
制作者: 黒天

-------------------------------------------------------------------------------
１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.285

■ 導入
下記の通り、ERBフォルダの配下に
同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

[フォルダ構成図]
+[ERB]
|
+--+[SYSTEM_FUNCTION.ERB]

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■ INPUT_YN関数を使用している二択選択肢でスペースキーを押した場合に"0"(はい)が
選択されるようになります。

(作者コメント)
面倒だったのでFPS対応の時みたいにフラグ関係なく有効にしちゃったけど
FLAG:二択入力設定フラグの値で有効・無効切り替えたほうがよかったかしら？

-------------------------------------------------------------------------------
３．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法
本パッチではフラグや関数の追加は行っておりません。

前の処理に戻したい場合はお手数ですが
0.285を別の場所で展開し、該当箇所の差分を比較し修正してください。
