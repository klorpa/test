○何するパッチなの
　P4隠しボスのマーガレットを捕獲するためのイベントを追加します。
P4では4人がかりでリンチだったけどeraMegatenでは6人がかりでリンチ。
だれだBBAとか言ったの！長鼻を歌う素敵な天然お姉さんだろ！

○当て方は？
　ver0.284+修正ファイルに当てれば問題ないはず。

○イベントの条件は？
　エリザベス撃破+P4系イベントの進行、もしくはデバッグフラグが立っていること。
「群青色の招待状」->「天国の最果てにて」が追加されます。
選択肢で二度と出ないようにフラグを折ることができます。
戦うつもりがなくて鬱陶しい時はさっさと折ってしまうといいでしょう。

○イベントの内容は？
　全能のバランサー(Lv68)+マーガレット(Lv74)との戦闘。
・全能のバランサー
　コンセントレイトを使用した後に6種の属性魔法のなかで弱点持ちを探して放ちます。いなければランダム対象。下手に削った状態で放っとくと(コンセ)メギドラオンがとんできます。
・マーガレット
　パターン完全固定、無効も反射も吸収もバステも知ったこっちゃない火力ごり押しキャラ。
六段階の行動パターンを持ち、毎ターン最初にステータスや耐性が変化。
行動パターンなどの予備知識なしだと若干厳しいです。
ノーマルであればエリザベスより弱いのでイベント条件的には倒せるはず。
50ターン毎に即死級威力のメギドラオンで攻撃してきますが、50ターンかかることなんて滅多にないです。
倒せたらお持ち帰りするかの選択があります。

○マーガレットさん弱すぎ(強すぎ)
　詳しい戦い方、というより調整中の戦い方はEVENT_16_AI.ERB内コメントに記載。
戦闘難易度ノーマルなら平均Lv60でも十分勝てるのを確認。
耐力を早めに上げてメギドラオンを耐えられるHP、マーガレットより先に攻撃できる速度を確保していればもっと低レベルでも倒せるはず
以前より弱点も増えてパターンと属性相性を覚えていれば弱いくらい
