*******************************************************************************
eraメガテン_仲魔悪魔敵出現時名前色変更パッチ
*******************************************************************************

■ 始めに
当パッチは仲魔にいる悪魔が敵で出現した場合に名前の色をピンクに変更するパッチです。


■ 内容物
・readme        (当ファイル)
・FORMATION.ERB (一部修正)

■ 謝辞
eramaker、Emuera、及びeraMegatenに関わったすべての方々への多大なる謝辞をこの場で申し上げさせて頂きます。
また該当モジュールの制作者様にも重ねて謝辞を申し上げさせて頂きます。
旅人様のreadmeの書き方がとてもすばらしかったためパk..参考にさせていただきました。
制作者: 黒天

-------------------------------------------------------------------------------
１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.282+修正ファイル

■ 導入
下記の通り、ERBフォルダの配下にある「XXXXX」配下の
対象フォルダ内に、同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

[フォルダ構成図]
+[ERB]
|
+--+[RPG]
   |
   +--+[セットアップ関連]
      |
      +--+[FORMATION.ERB]

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■ 仲魔にいる悪魔が敵として出現した時に名前をピンクで表示します。
それだけです。

(作者コメント)
狩場に仲魔悪魔がいるとき属性調整で時々TALKしないといけないんだけど、
「仲」だけだといるかどうかの判断にちょっと時間がかかってたので色を変えてみた
俺以外の誰得でもないパッチかと思うけどせっかく作ったので公開してみる。

-------------------------------------------------------------------------------
３．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法

;---- EDIT 005 MOD START ---------
	CALL TEST_ABC
;---- EDIT 005 MOD END -----------

「EDIT 005」等で検索すれば、手を加えた全ての修正箇所が分かります。
前の処理に戻したい場合はお手数ですが
0.282を別の場所で展開し、該当箇所の差分を比較し修正してください。
