■ 始めに
当パッチは食堂・売春などの労役に関するシフト設定において
複数のキャラのシフトを一括で設定できるようにするパッチです。


■ 内容物
・readme   (当ファイル)
・WORK.ERB (一部修正)

２．機能
-------------------------------------------------------------------------------
■ 労役のシフト設定でシフト設定キャラの選択画面に
[999] 全員のシフト設定を変更する。を追加
上記項目を選択した場合、シフト設定が可能な全員のシフトを同じ設定に変更することが
できます。
なお、上記項目を選択した場合、シフトの異なる各キャラのシフトを統一するため
設定前に全員のシフト設定が毎日ON(勤務状態)に変更されます。

(作者コメント)
食堂労役修正パッチでシフトの重要性が高まり、各キャラいちいち設定するのは
面倒だったのでキャラ引継ぎに続き一括処理を作りました。
全キャラ同じパターンで登録されてしまうので、２・３パターンぐらいシフトを
作りたい人は同じパターンのシフトにしたいキャラ以外を一旦労役解除し、
シフトを設定後、労役しているキャラの労役を解除、次のパターンで労役したいキャラを
労役させてシフト設定〜を繰り返してください。

３．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法

;---- EDIT 003 ADD START ---------
	CALL TEST_ABC
;---- EDIT 003 ADD END -----------

「EDIT 003」等で検索すれば、手を加えた全ての修正箇所が分かります。
前の処理に戻したい場合はお手数ですが
0.284を別の場所で展開し、該当箇所の差分を比較し修正してください。
