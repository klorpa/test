ＶＲ美術館追加試作　20110814　(´・ω・｀)

スナッピーが瀕死になった時に取引を持ちかけてくるようにしてみました。
受け入れると美術館内のイルカ部屋で肉便器の使用が可に。
使用時はアライメントがＤ方向へ変動の後、強制脱出します。
拒否すれば従来通りの展開になる…んですが、倒し切れないと何度も命乞いしてきます…。
後、取引成立後にも最後に一発殴ってきます。上手い動作のさせ方が分らなかったのでそのまま…。
使ってみようという方は、ＶＲホラーハウス側の出現条件の20行目を
SIF (DB:90:1 >= 3 && DB:92:0 == 0) || DB:92:0 == 1 || FLAG:DEBUG
にお手数ですが変えておいて下さい。
