eraメガテン_SJ修正パッチ

■ 始めに
当パッチはシュバルツバースで発生しているいくつかのバグの対処および
自分の好みの問題でいくつかの機能追加・修正を行うためのパッチです。


■ 内容物
・readme                               (当ファイル)
・DUNGEON62_セクター・カリーナ.ERB     (一部追加)
・DUNGEON63_セクター・ボーティーズ.ERB (一部修正)

■ 謝辞
eramaker、Emuera、及びeraMegatenに関わったすべての方々への多大なる謝辞をこの場で申し上げさせて頂きます。
また該当モジュールの制作者様にも重ねて謝辞を申し上げさせて頂きます。
旅人様のreadmeの書き方がとてもすばらしかったためパk..参考にさせていただきました。

-------------------------------------------------------------------------------
１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.292 + 修正ファイル4

■ 導入
下記の通り、ERBフォルダの配下にある「RPG」配下の
対象フォルダ内に、同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

[フォルダ構成図]
+[ERB]
|
+--+[RPG]
   |
   +--+[ダンジョンアタック]
      |
      +--+[ダンジョンデータ]
         |
         +--+[DUNGEON62_セクター・カリーナ.ERB]
         |
         +--+[DUNGEON63_セクター・ボーティーズ.ERB]

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■ バグ修正
１．ダンジョンから脱出できるように運転手追加
２．セクター・ボーディーズ内の敵選択処理内で旧敵選択処理が中途半端に
　　実装されていたためSET_ENEMYで無限ループしていたのを修正
３．セクター・ボーディーズから船に戻ると壁の中にいるのを修正

■ 機能追加・修正
４．調査船内時のダンジョン名表示で現在いるセクターの表示を追加
５．前に出したセクター間移動処理パッチの内容が入っていなかったので追加
６．セクター・ボーディーズから船に戻るマスをセクター・カリーナと同様に「船」に変更


-------------------------------------------------------------------------------
３．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法
本パッチではフラグの追加などは行っておりません。

修正箇所については0.292+修正ファイル4環境を別の場所に展開し、本パッチに含まれている
ファイルと同名のファイルと内容を比較して確認してください。
前の処理に戻したい場合もお手数ですが
0.292+修正ファイル4の環境を別の場所に展開し、同名ファイルの差分を比較し修正してください。
