【はじめに＆諸注意】**************************************************************
　この度は、本口上をダウンロードしていただきありがとうございます。
　Zoma口上（３）、Zoma様口上になります。
　「てめぇはホントにZomaか！？」と言わせるのがコンセプト……なんですけど、まあ、人を選ぶかも……
　まあまだ調教開始・終了しか書いてませんけどねー。
　自分的には0.2くらい。方向性で0.1、開始終了で0.2、ってくらい。

　「あー、600ｋｂ越えたし、ちっと別口上にも手を出してみるか……」
　「んー、ポンコツZoma口上出たしアレマジ可愛いし泣かしてえし売りてえしZomaたん構想はポイしていいかなー。じゃあこれだ」　
　とやってきたネタをフォイフォイしてみました。
　とりあえず、殴って言うことを聞かせてあげたくなるよーな感じになってれば幸いなんですけれど。
　ドSフォイフォイを目指して頑張ります。
　セレクタ入ってるのでとりあえずぶちこんでも容量の無駄以下にはならないはず。

　構文ミス修正、日本語修正、加筆なんでもどんとこい。来てくれ。俺マジ勉強とかしてないから。
　元々からして口上加筆者・統合者なので、パーツ一個からでもうｐして頂ければテキトーに組み込みます。どんとこい。

【コンセプト】**************************************************************
●『Zoma“様”』
　無駄に偉そう。
　基本的に上から目線。
　しかし人生経験少ないんで逆らわれるとビビリ入る。
●『虚心』
　喋っていいことの基準がないため口数が多い。
　また、基本的に後ろ向き。自分に自信がないため、常に己を鼓舞するような行動・口調でいる。
　根っこと呼べる部分が存在しないため、周囲に攻撃的になっている。
　感情豊かなように見えて、『高圧的』な一面しかない。
●『反発刻印ついてる方が元気』
　現在のところはそこまで書いていないですが、『とにかく土下座とか涙目とかにしたくなる』感じで。
　生意気っ子が涙目ダブピービデオレターでバージンロスト、みたいなー？
●各ルートについて
　恋慕：偉そう。さあメイクラブしてあげてもいいよ？
　淫乱：偉そう。気持ちいいなら奉仕してあげてもよいのだよ？
　服従：ﾊﾞﾝｯ　ひっ
●地の文少なめ
　口数がかなり多い感じで。
