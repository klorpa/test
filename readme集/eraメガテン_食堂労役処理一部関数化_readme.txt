■ 始めに
当パッチは食堂労役に関する処理の一部を関数として分散、および
使用LOCAL変数の見直し、その他多少の修正を行ったパッチです。
実際の処理内容に変更はありません(はずです)。


■ 内容物
・readme       (当ファイル)
・RESTRANT.ERB (一部修正)

eramaker、Emuera、及びeraMegatenに関わったすべての方々への多大なる謝辞をこの場で申し上げさせて頂きます。
また該当モジュールの制作者様にも重ねて謝辞を申し上げさせて頂きます。

-------------------------------------------------------------------------------
１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.294+修正ファイル

■ 導入
下記の通り、ERBフォルダの配下にある「調教関連」配下の
対象フォルダ内に、同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

[フォルダ構成図]
+[ERB]
|
+--+[調教関連]
   |
   +--+[労役]
      |
      +--+[RESTRANT.ERB]

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■ 修正内容
・労役単価計算処理をSALARY_RESTRANT関数に一本化、第二引数で
　計算情報の表示有無を切り替えられるように
・難易度補正、引継ぎキャラ補正は他の労役との共通部分なので関数化して分離してみる。
・NTR処理も結構ボリュームが大きかったので関数化して分離。
・各関数で使用するLOCAL変数の見直し、LOCALSIZEの設定
・就労決定時のメッセージが一日限りっぽい表示だったので変更
・NTR処理で素質[Male]がない場合の処理の中で
　素質[Male]の有無をチェックしていたので削除

-------------------------------------------------------------------------------
３．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法
本パッチではフラグの追加などは行っておりません。

修正箇所については0.294+修正ファイル環境を別の場所に展開し、本パッチに含まれている
ファイルと同名のファイルと内容を比較して確認してください。
前の処理に戻したい場合もお手数ですが
0.294+修正ファイル環境を別の場所に展開し、同名ファイルの差分を比較し修正してください。
