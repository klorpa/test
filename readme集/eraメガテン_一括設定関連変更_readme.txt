■ 始めに
当パッチは「キャラ引継ぎ」「食堂労役」「売春労役」「シフト設定」における
一括設定処理の動作をある程度統一し、「労役解除」で一括選択を可能にするためのパッチです。


■ 内容物
・readme               (当ファイル)
・SHOP_NEWGAME.ERB (一部修正)
・WORK.ERB         (一部修正)
・PROSTITUTION.ERB (一部修正)
・RESTRANT.ERB     (一部修正)

１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.288+修正ファイル4

■ 導入
下記の通り、ERBフォルダの配下にある
対象フォルダ内に、同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

[フォルダ構成図]
+[ERB]
|
+--+[ＳＨＯＰ関連]
|   |
|   +--+[SHOP_NEWGAME.ERB]
|
+--+[調教関連]
   |
   +--+[WORK.ERB]
   |
   +--+[労役]
      |
      +--+[PROSTITUTION.ERB]
      |
      +--+[RESTRANT.ERB]

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■
・「キャラ引継ぎ」「食堂労役」「売春労役」「シフト設定」における動作を以下に統一
  - 対象キャラが二人以上いる場合にのみ一括選択の選択肢を表示
  - 一括選択に対応する番号は[999]
  - 一括選択肢が出ていない場合、直接番号を入力しても無効に
  - 間違って押した場合に備え、本当に一括処理を行うかの確認を実施。
・「労役解除」に一括選択の選択肢を追加
・「労役解除」でキャラの労役解除後、選択時に表示していたページを表示するよう変更
  （2ページ目のキャラの労役を解除した場合、その後のキャラ一覧で2ページ目を表示）

-------------------------------------------------------------------------------
３．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法
本パッチではフラグの追加などは行っておりません。

修正箇所については0.288+修正ファイル4環境を別の場所に構築し、本パッチに含まれている
ファイルと同名のファイルと内容を比較して確認してください。
前の処理に戻したい場合もお手数ですが
0.288+修正ファイル4の環境を別の場所で展開し、該当箇所の差分を比較し修正してください。
