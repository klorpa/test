■ 始めに
当パッチは仲間強化での耐性強化において核熱耐性の強化ができない
問題についての対処を行っています。。


■ 内容物
・readme  (当ファイル)
・MAG.ERB (一部修正)

■ 謝辞
eramaker、Emuera、及びeraMegatenに関わったすべての方々への多大なる謝辞をこの場で申し上げさせて頂きます。
また該当モジュールの制作者様にも重ねて謝辞を申し上げさせて頂きます。
旅人様のreadmeの書き方がとてもすばらしかったためパk..参考にさせていただきました。

-------------------------------------------------------------------------------
１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.281R

■ 導入
下記の通り、ERBフォルダの配下にある「女神転生」配下の
対象フォルダ内に、同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

[フォルダ構成図]
+[ERB]
|
+--+[女神転生]
   |
   +--+[MAG]
      |
      +--+[MAG.ERB]

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
仲魔強化の項目「耐性強化」において核熱相性の強化を選択しても強化内容の
選択画面に遷移しない問題を対処に正常に核熱相性の耐性強化を行えるように対処

-------------------------------------------------------------------------------
３．修正内容
-------------------------------------------------------------------------------
MAG.ERBの875行目の内容を以下のとおり変更しています。
(行数は環境によって変わる可能性があります。)
修正前）ELSEIF RESULT < 0 || RESULT >= FLAG:相性数-2
                ↓                                 ~
修正後）ELSEIF RESULT < 0 || RESULT >= FLAG:相性数-1
                                                   ~
上記以外の変更はありません。
