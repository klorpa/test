eraメガテン_FPS操作時戦闘コマンド対応キー変更パッチ
*******************************************************************************

■ 始めに
当パッチはFPS操作時にAUTO系のコマンドを実行する際にテンキーの数字を打つのが面倒な方用に
FIGHT、TALKおよびAUTO系コマンドを実行するためのキーを変更するパッチです。


■ 内容物
・readme     (当ファイル)
・BATTLE.ERB (一部修正)

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■ ダンジョン内操作設定が「ＦＰＳ」の場合にコマンドに対応するキーを以下の通り変更します。

コマンド    |通常時|ＦＰＳ時
FIGHT       | [0]  | [ ](SPACEキー) ←追加
TALK        | [1]  | [w]
AUTO        | [4]  | [a]
GUN AUTO    | [5]  | [s]
REPEAT AUTO | [6]  | [d]

実際は通常時だろうがＦＰＳ時だろうがどっちのキーでも同じ動作になります。
通常時に"w"打ってもTALKになるし、FPS時に"4"を打ってもAUTOになる。
あと" "(SPACEキー)でもREPEAT AUTOになったりしてます。

(作者コメント)
他者の操作感覚が分からないので感想をいただけるとうれしいです。
このコマンドをこのキーに割り当ててほしいなどの要望
その他ご意見がございましたらスレにて書き込みをお願いします。

【2011/02/13修正】
・SPACEキーの割り当てをREPEAT AUTOからFIGHTに変更
・FIGHT後の入力画面でのSPACEキーでGO(戦闘開始)になるように修正(既存の0でももちろんOK)
　ただし、入力モードがキーボード時のみ有効
