*******************************************************************************
eraメガテン_調査船に戻る対応
*******************************************************************************

■ 始めに
当パッチはSJダンジョンのターミナルの選択肢「調査船に戻る」を有効にするためのパッチです。

■ 内容物
・readme                           (当ファイル)
・DUNGEON62_セクター・カリーナ.ERB (一部修正)

■ 謝辞
eramaker、Emuera、及びeraMegatenに関わったすべての方々への多大なる謝辞をこの場で申し上げさせて頂きます。
また該当モジュールの制作者様にも重ねて謝辞を申し上げさせて頂きます。
旅人様のreadmeの書き方がとてもすばらしかったためパk..参考にさせていただきました。

-------------------------------------------------------------------------------
１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.285

■ 導入
下記の通り、ERBフォルダの配下にある「RPG」配下の
対象フォルダ内に、同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

[フォルダ構成図]
+[ERB]
|
+--+[RPG]
   |
   +--+[ダンジョンアタック]
      |
      +--+[ダンジョンデータ]
         |
         +--+[DUNGEON62_セクター・カリーナ.ERB]

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■ 
セクター・カリーナおよびセクター・ボーディーズのターミナルで
「調査船に戻る」を選択した場合に船に戻るようになります。
あと、ボーディーズから戻った後、カリーナを探索するとボーディーズから始まるので
完全じゃないかもしれないけど対処してあります。


-------------------------------------------------------------------------------
３．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法
本パッチではフラグや関数の追加は行っておりません。

前の処理に戻したい場合はお手数ですが
0.285を別の場所で展開し、該当箇所の差分を比較し修正してください。
