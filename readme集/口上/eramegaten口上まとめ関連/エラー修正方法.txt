頻出っぽいエラーについて

□警告Lv1:XXX.ERB:Y行目関数@ZZZは既に定義(AAA.ERBのB行目)されています
	名称やフォルダ構成が変更された場合、旧ファイルが残ってると大量に出ます
	旧ファイルを削除すると解消することができます
	というか、性質上、口上まとめは口上ファイルを一旦全て削除してからあてることを推奨します

□警告Lv2:XXX.ERB:Y行目:'%'が使われましたが対応する'%'が見つかりません
	%が足りてないか、多いかです
	適切に補うことにより解消可能です

□キャラクタ配列変数NOの第１引数(-1)はキャラ登録番号の範囲外です
	NO:-1などはエラーを起こしておちます
	特にASSIは不在である場合、-1なのでNO:ASSI == [[キャラ:××]]などをやるとかなりの確率でエラー落ちを引き起こします
	IF ASSI > 0 && NO:ASSI == [[キャラ:××]]
	といったように記述すれば、1つ目の条件を満たした時、2つ目の条件を判定しないのでエラーを回避できます

□ゲーム中でCALLNAME:TARGETなどといった表記がでてくる
	FORMをつけ忘れてます
	GREPでPRINTL/PRINTW をやったあと%で検索してみると見つかったりします

□警告レベルについて
	本体を起動し、ヘルプで解析タブ
	ロード時に引数を解析する→常に行う
	表示する最低警告レベル→0.標準でない文法

	これでかなりエラーひっかけられるんでアップする前にチェックしてみるといいと想います
