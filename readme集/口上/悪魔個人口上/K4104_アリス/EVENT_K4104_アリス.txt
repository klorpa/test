●使用上の諸注意
エロさより作者がイメージする「アリスらしさ」を重視しています。
現在埋まっている部分はほぼ恋慕・親愛のみになっています。
一部汎用口上をそのまま流用しています。
男「あなた」を想定しています。

●コンセプト
作者がイメージする「アリスらしさ」について箇条書きしていきます。
	・モットーは『楽しいのが好き、寂しいのは絶対に嫌』
	・理想の男性＝『王子様』
	・Not ヤンデレ

●ギミック『鏡の魔術』
『デビルサマナー葛葉ライドウ対コドクノマレビト』に登場したアリスの魔術です。
アリスに素質【Affection】、または【Love】が付いている状態で[186]羞恥プレイを実行すると「3分の1の確率」で発生します。
発動中は一部口上の文章が追加・変更されます。
アリスの申し出を拒否した場合、通常の羞恥プレイ扱いとなりますが、許可した場合は羞恥プレイ状態はキャンセルされます。
また、初回実行時に2000円消費されます。
解除するためには調教を終了する必要があります。
