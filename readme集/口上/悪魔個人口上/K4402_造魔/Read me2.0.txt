【はじめに＆諸注意】**************************************************************
　本口上をダウンロードしていただきありがとうございます。
　今回は新テンプレへの移行がメインです。
　テンプレ移動のときにちょこちょこ加筆したりしてたんですが……まあ、最近忙しかったので、って話でして。増えたｋｂはだいたいテンプレ分のはず。
　ってか間が空いたから自分でもよく分からねぇ。

　口上ろだ306の上司イベント加筆を取り込みさせていただきました（感謝）。
　その際、スペースでインデントってあった部分修正したり文章を改変したりなんだりでわりと別物になってる気がしますけど。
　本体に取り込まれてはいないのでコメントアウトしてあります。

　戦闘口上について。
　一応かなり充実はしていると思いますが、まあ、ネタ多数なんですよね。
　他作品ネタ使うのはあんまりいくねえよなー、とは思うんですけど、書いたの消すの惜しいんだモンッ！
　まあ基本RAND確率低くしてあるんでERB覗かん限り中々見れないと思う。
　あとあの辺ホントコメント多くてごめんなさい。美しくないですよね。

　構文ミス修正、日本語修正、加筆なんでもどんとこい。来てくれ。俺マジ勉強とかしてないから。
　元々からして口上加筆者・統合者なので、パーツ一個からでもうｐして頂ければテキトーに組み込みます。どんとこい。
　とりあえず口上フォルダにぶっこめば動くと思いますが、「こんなんウチのZomaちゃんじゃない」ってのにも備えて、バックアップ推奨です。
　デレッデレなのはコメントアウトしてるので、一通り遊んだらのぞいてみてくださいな。多分、『デレ』で検索したら出ると思います。
　『ネタ』でも色々出る気がしますが。

【ライセンス】**************************************************************
　EVENT_K4402_Zoma_1_ALL.ERB参照。
　ライセンスフリーです。
　自分の加筆部分のライセンスについても、作者様（口上ろだ104のりどみ参照）のそれに従い、
　”私（NEO）の製作物（加筆部分）は全てライセンスフリーです。加筆も改変も流用も可です。そぐわないようでしたら、該当箇所の削除も可です。”
　……と明言しておきます。
　将来的に自分みたいのが出て、さらなる加筆があったなら、それはとっても嬉しいなって。
　改変不可にしたらそういうモノ好きの道が断たれてしまいますしー。そもそも細かい加筆とか結構多くて自分の加筆した部分だけ禁止とか面倒すぎますしー。

　今度こそ各種コマンドの加筆。

　２、奉仕系

　……と、順番に。締め切りはありませんが、Zomaちゃんのためなるべく早く頑張る、と言ったところで。
　育て上げて鍛えきったところで出るものや、よく使うもの（＝自分でよく見るもの）から優先的に埋めていこうかな、と。
　また絶頂口上の方も、セックス系など、コマンド口上との兼ね合いで色々改変しようかと思ってます。

　将来的には、素質関連で色々分けたい。巨乳貧乳、ちんちんの有無、敏感の有無から、臆病や好奇心まで。
　愛撫とか結構分かれてますけど、アレやったの大体俺じゃねえし、他も愛撫くらいとまでは言わずとも、色々分けたい。
　Zomaはかなりカスタマイズできるんで、その辺色々……[恥じらい]や[中毒しやすい]とか楽しそうですよね。
