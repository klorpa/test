移植元のJの使用上、恋慕、親愛しか存在しません、

使用方法
ERB →　口上　→　悪魔個人口上のフォルダーに入れてください。


以下、移植元の read me

このファイルには、Ｊ用紫口上、永琳口上、豊姫口上、依姫口上が入っています
エンディング部分が加筆されましたので、最後まで使って頂けると有り難いです
これらの口上は、他多数の口上およびたくさんのパッチを参考にして作られています
この場を借りて、各口上製作者並びにパッチ製作者の方々にお礼を申し上げます

もし加筆したいという方がいらっしゃいましたら、お好きなように書き換えて構いません
他の口上に利用したいという場合は、
・シチュエーションの流用など　　　　　→　可
・構文の利用など　　　　　　　　　　　→　可
・キャラ名を変えただけのまるまる移し　→　不可
という形になっております

だいたいの利用などは可能ですが、まるまる写すのは不可という認識であれば大丈夫だと思います
何かありましたら、連絡してくれるとありがたいです。
スレや口上ｃｈなどであれば反応出来るかと思います




一応、CFLAGの使用状況などをまとめた資料を同封しております

〜各種口上について〜
【紫口上】
自分が書いた紫口上は、どちらかというとＭ側に寄ってるらしいです
Ｓ側の紫じゃなきゃ嫌だという方は、使わない方が賢明だと思われます
そうでなくても質の低い口上である為、合わないと感じたら使用を一時中断して、中の人（空っぽ）様やゆかてん様の書いた口上などで抜いてくるのがいいと思います

【永琳口上】
薬を使って猫耳になったりはしません、ご了承下さい
処女分岐が一部にありますので、よろしければ最初の選択肢で処女にしてみて下さい

【豊姫口上】
なんかよく分からないキャラに仕上がってるような気がします
何か変だな、という部分がいくつかあると思いますが、最後まで使って頂けるとありがたいです

【依姫口上】
他の方が作った、ツンデレよっちゃん口上は質が高いです。
こちらの口上はあんまりデレないかもしれませんが、最後まで楽しんで頂けるとありがたいです
