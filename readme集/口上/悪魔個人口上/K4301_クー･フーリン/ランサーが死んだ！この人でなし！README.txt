RKSI様のクー・フーリンの某運命槍兵風口上改変版。RKSI様と元型となったジャンヌ口上作者様には多謝いたします。
経験不足と実力不足もあってあちらこちらからコピペもとい参考にしたために稚拙な部分も多いです。
なおこの口上には特定のコマンドを使用した場合において調教の趣旨とは大きく外れた口上テキストが存在し、プレイヤーにMasoEXPが付く場合が存在します。
そういうこともあって「この口上ダメじゃね？ゴミ箱にぶちこんじまえ！」と思ったら迷わず実行して頂いて構いません。
