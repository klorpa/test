パッチの適用砲
ERBフォルダをぽいっとやってください。

適用するとHoneyBeeをONにしている場合にダンジョンメニューのCOMPのメニューにマップ表示ってのが追加されます。
でそいつを選ぶとマップが表示されるわけですが
HoneyBeeは登録されたマップ番号に従ってマップを表示します。
なので病院とかを見るとわかりやすいかと思いますが、階層ごとの表示ではありません。
何か面白い方法があればきちんと階層ごとにつなげて表示とかするかもしれませんが、予定は未定です。
