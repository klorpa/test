*******************************************************************************
eraメガテン_公衆便所拡張規模指定パッチ
*******************************************************************************

■ 始めに
当パッチは公衆便所の雇用人数および利用可能客数の拡張において
複数人分一気に追加できるように変更するパッチです。
ついでに宣伝広告でパラダイムXと飛行船を選択したときのメッセージが
ラジオと一緒だったのを修正しています。

■ 内容物
・readme           (当ファイル)
・MANAGEMENT.ERB   (一部修正)

■ 謝辞
eramaker、Emuera、及びeraMegatenに関わったすべての方々への多大なる謝辞をこの場で申し上げさせて頂きます。
また該当モジュールの制作者様にも重ねて謝辞を申し上げさせて頂きます。

-------------------------------------------------------------------------------
１．導入方法
-------------------------------------------------------------------------------
■ 対象環境
ver0.285

■ 導入
下記の通り、ERBフォルダの配下にある「調教関連」配下の
対象フォルダ内に、同封のERBファイルを配置・上書きして下さい。
(元ファイルがある為、上書きを確認されると思います。)
既に当該ファイルを改造されている場合には、差分導入をお願いします。
また環境が違う場合も不具合が発生する恐れがある為、こちらも差分導入をお願いします。

[フォルダ構成図]
+[ERB]
|
+--+[調教関連]
   |
   +--+[労役]
      |
      +--+[MANAGEMENT.ERB]

-------------------------------------------------------------------------------
２．機能
-------------------------------------------------------------------------------
■ 
・公衆便所の便器数の拡張および利用可能客数を既存の一人ずつ追加から
　1、5、10人分まとめて拡張できる選択肢を追加
　また上記選択肢以外にも直接人数を指定することで指定人数分の拡張を行えるように変更

・広告宣伝においてパラダイムX、飛行船を選択した場合にラジオ選択時と
　同じメッセージが表示されていたのを修正 
-------------------------------------------------------------------------------
３．開発者様へ
-------------------------------------------------------------------------------
■ 修正工法
本パッチではフラグの追加などは行っておりません。
関数「@規模拡張金額(ARG,ARG:1,ARG:2)」を追加しています。
詳しい関数の説明についてはMANAGEMENT.ERBの該当関数に書かれているコメントを参照してください。

前の処理に戻したい場合はお手数ですが
0.285を別の場所で展開し、該当箇所の差分を比較し修正してください。
