
>>古代ちゃんは口上制御で、友好的に加入してもらった場合忠誠度が一定以上だと勝手に恋慕状態になって、
>>初調教時に恋慕状態だと和姦っぽくなって戦闘参加不能にならないみたいなのを仕込もうかと

てのを、調教しなけりゃ戦闘手伝ってくれる人間キャラとかいっそ悪魔にまで適応できるようにするオプションを追加
誤解させたまま惚れさせて、調教をすんなり受け入れさせるとかいいよね。

・オプションを1にすると、CFLAG:戦闘参加不可能=-1の仲魔（主にイベント加入の人間）を、2にすると全ての仲魔を対象に取ります
　忠誠度を10000〜40000（属性値・性別により変化）以上にする・１度も調教や性的な労役をしてない・CFLAG:NTRフラグ=0の全てを満たすと
　１日の終りに従順３・屈服刻印３・恋慕・CFLAG:未調教陥落=1がつきます。
　口上作るときはそこで分岐させることができると思います。テンプレや既存口上にも追加してるんで参考にしてください

　なお、基本１万＋属性の違い（LND/LNC）１ずつにつき+2500、主人と同性ならさらに倍（おまけに主人に謎の魅力が必要）って計算なんで
　大抵は10000〜20000の範囲に収まるレベルかと思います。謎の魅力持ち主人だと20％引き

・２身合体で作成された悪魔の場合、素質変化や能力継承を終えた後の忠誠度を０として、そこから増加した忠誠度で判定します
　娘は最大で2000、周回引き継ぎ仲魔（陥落消し）は最大で5000、必要な忠誠度が軽減されます

・ユミとたまきちゃんも誤解させたまま攫ってこれるよね、てことで対象に。単なる趣味です

・他の幾つかの口上で、調教されたの前提な恋慕口上を適当かつ元のを参考に、和姦っぽくなるよう分岐させてみる。
　どっちも暫定版なんで加筆改変オールオッケーばっちこい。改変場所は"暫定"で検索してください
・幾つかの口上の、はい/いいえの選択肢をINPUT_YNに置き換える

・りせ口上の開始時、苦痛刻印で分岐するところをTFLAG:22（苦痛刻印取得フラグ、開始時は常に0）で分岐してたので修正
・星あかり口上の開始時、主人の陥落を参照してたところを修正
・古代口上に、トールとリンクしてて電撃相性スキルを使用したときの口上追加。とりあえずミョルニルって叫ぶのが古代ちゃんクオリティだと思うんだ

・性格による忠誠度上昇をシステム側に配置しなおし。忠誠度の上がる行動が無かった以下の性格に上がる条件を追加。我ながらややこしい
　高慢：スキルタイプ＝3状態変化　天使：相性＝破魔　堕天使：相性＝呪殺
　賢い：弱点相性を付いたor行動不能な状態異常の敵に対する攻撃
　残虐：高威力・広範囲の攻撃。物理と魔法でスキルの威力差が大きいので、計算式も別に
