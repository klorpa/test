4.派生コマンド
	派生系って隠しコマンドもいいところすぎるんで、このくらいのヒントはでたほうがいいんじゃないかと思ったりしたので

	名称だけでなく、コマンド番号も渡している
	なので、@EVENTCOMに来る前の段階で、すでにSELECTCOMに派生コマンドが渡されてる
	これで@EVENTCOMで色々する必要ができたときに便利！…かもしれない

8．３Ｐ
	派生元によって内容が変わるコマンドなので、困った気分に
	とりあえず、上記の３種類の名前に分割しましたが、名称についてはわりと悩む感じに
	あと、誰かのＶ/Ａに挿入してるのにたいして、挿入するとか
	主人が両穴させるとか、そういうのもありなんじゃとおもったけど、ひとまずおいておきます
	
	TEQUIP:X:◯挿入者は、Xの◯に挿入しているキャラを示しています
	TFLAGじゃないのは拡張性を考えて、TCVARじゃないのは添字の問題です
	これでだいぶ3Pの記述自体はすっきりしたんじゃないかと

	基本的に記述整理で、助手のソースとか、MAG移動についての措置の変更とか
	実内容で影響ありそうな部分は触ってません

	助手・奴隷ともに体力の消耗は激しくなるけど、両方同時に調教（MAG移動）できるような感じ
	もありなんじゃないかと思うけど、まぁとりあえずおいておきます

10.調教メニュー
	CALLTRAINを投げ捨てた
	まぁ、CALLTRAINは、普通のCOM実行と同じ経路を通るのが売りで
	その経路を破壊している以上は、DOTRAINで手実装していいかなぁと

	手実装したうえでのメリットとしては、主人と助手を指定して登録可能になったり
	調教メニューにたいして、コマンドを挿入することも可能になったので
	逆調教で、自動でCOM走った際に、CALLTRAINされてても、そのまま走らせられるとか、中断を選べるとかまぁそんなこともできたり
	逆調教でDOTRAIN関係しようとした場合、この実装のほうが困らないはず…
	ですがまぁ、助手とか奴隷が勝手になんかするとか、実際に作るかまではちょっとよくわからないです

	登録できる数を5個にしました
	メニューに名前をつけられます
	そして、グローバル管理にしました
